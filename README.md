# Hotel

*Sistem za rezervacijo hotelov:*

Opis teme mojega izdelka::

Tema mojega izdelka je sistem za rezervacijo hotelov, ki je namenjen enostavni uporabi da uporabnik preprosto uporabi program, kjer se prijavi in si rezervira bivanje, kjer si izbere hotel, sobo, obdobje bivanja in potrdi rezervacijo. Vpisujejo pa se lahko tudi zaposleni in se shranjujejo in zapisujejo.







Navodila za namestitev in poganjanje::


1. Uporabnik mora imeti nameščen Visual Studio v katerem bo odprl program
2. Uporabnik mora v brskalniku odpreti GitHub in naložiti program iz repozitorija
3. Potem mora odpreti program v Visual Studiu in ga zagnati čisto na vrhu aplikacije
4. Ko vse to opravi lahko začne z uporabo programa in začne z "registracijo" in izbiranje bivanja.







Navodila za uporabo::


1. Po zagonu programa se prikaže celoten grafični vmesnik
2. Uporabnik mora najprej vpisati svoje osebne podatke in izbrati ali je gost ali zaposlen, saj če je gost mora dodatno vpisati še svojo številko osebnega dokumenta
3. Po vnašanju podatkov mora izbrati datum od kdaj do kdaj želi bivati, v katerem hotelu, katero sobo saj jih je več z različnimi cenami in si izbere tisto sobo ki mu odgovarja.
4. S pritiskom na gumb Rezerviraj se rezervacija shrani v desni stolp.
5. Potem pa če se kakšna rezervacije prekliče se lahko iz stolpa desno tudi izbriše in je vse bolj pregledno.
